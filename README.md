
# Aneurysm Detection using Deep Learning

## Overview
This project detects brain aneurysms from medical images using a CNN (Xception).

## Dataset
https://www.kaggle.com/datasets/trainingdatapro/computed-tomography-ct-of-the-brain
- Medical imaging dataset (CT/MRI)
- Preprocessed with normalization and resizing

## Model
- Xception-based CNN
